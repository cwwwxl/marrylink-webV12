# 用户管理用例描述表

## 用例规约

---

### 用例编号：UC-001
### 用例名称：登录系统
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员、新人、主持人 |
| **用例描述** | 用户通过账号密码登录系统，获取系统访问权限 |
| **前置条件** | 用户已注册账号 |
| **后置条件** | 成功登录后跳转至对应角色主页，显示用户信息 |
| **基本流程** | 1. 用户进入登录页面<br>2. 输入账号和密码<br>3. 点击登录按钮<br>4. 系统验证账号密码<br>5. 验证通过，跳转主页 |
| **备选流程** | 1. 账号不存在，提示"账号不存在"<br>2. 密码错误，提示"密码错误"<br>3. 账户已禁用，提示"账户已禁用"<br>4. 连续失败3次，锁定账户 |

---

### 用例编号：UC-002
### 用例名称：查询用户
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员 |
| **用例描述** | 管理员查看系统用户列表，支持筛选和分页 |
| **前置条件** | 管理员已登录系统 |
| **后置条件** | 展示符合条件的用户列表 |
| **基本流程** | 1. 管理员进入用户管理页面<br>2. 输入搜索条件<br>3. 选择用户类型筛选<br>4. 点击搜索按钮<br>5. 系统返回用户列表<br>6. 展示分页结果 |
| **备选流程** | 1. 无搜索条件，显示全部用户<br>2. 筛选条件无结果，提示"未找到相关用户" |

---

### 用例编号：UC-003
### 用例名称：新增用户
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员 |
| **用例描述** | 管理员创建新用户账号 |
| **前置条件** | 管理员已登录系统 |
| **后置条件** | 成功创建用户账号，发送账号信息给用户 |
| **基本流程** | 1. 管理员点击新增用户<br>2. 选择用户类型<br>3. 填写用户基本信息<br>4. 设置初始密码<br>5. 选择用户角色<br>6. 点击保存<br>7. 系统验证信息<br>8. 创建用户成功<br>9. 发送账号信息 |
| **备选流程** | 1. 信息不完整，提示必填项<br>2. 账号已存在，提示"账号已存在"<br>3. 验证失败，返回错误信息 |

---

### 用例编号：UC-004
### 用例名称：编辑用户
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员 |
| **用例描述** | 管理员修改用户信息 |
| **前置条件** | 管理员已登录系统，目标用户存在 |
| **后置条件** | 用户信息更新成功 |
| **基本流程** | 1. 管理员选择目标用户<br>2. 点击编辑按钮<br>3. 修改用户信息<br>4. 点击保存<br>5. 系统更新数据<br>6. 返回成功提示 |
| **备选流程** | 1. 修改账号时，新账号已存在，提示冲突<br>2. 修改敏感信息需二次确认 |

### 用例名称：删除用户
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员 |
| **用例描述** | 管理员删除用户账号 |
| **前置条件** | 管理员已登录系统，目标用户存在 |
| **后置条件** | 用户被逻辑删除，状态更新 |
| **基本流程** | 1. 管理员选择目标用户<br>2. 点击删除按钮<br>3. 弹出确认对话框<br>4. 确认删除操作<br>5. 系统执行逻辑删除<br>6. 返回删除成功 |
| **备选流程** | 1. 用户有待执行订单，提示"无法删除"<br>2. 取消删除操作，流程结束 |

| **后置条件** | 用户账户状态更新 |

### 用例编号：UC-007
### 用例名称：分配角色
| 项目 | 内容 |
|------|------|
| **参与者** | 管理员 |
| **用例描述** | 管理员为用户分配系统角色 |
| **前置条件** | 管理员已登录系统，目标用户存在 |
| **后置条件** | 用户角色关系更新，权限生效 |
| **基本流程** | 1. 管理员选择目标用户<br>2. 点击分配角色<br>3. 弹出角色列表<br>4. 勾选对应角色<br>5. 点击保存<br>6. 系统更新角色关系<br>7. 清空权限缓存<br>8. 返回成功提示 |
| **备选流程** | 1. 未选择任何角色，提示"至少选择一个角色"<br>2. 角色分配冲突，提示冲突信息 |

## 术语表

| 术语 | 定义 |
|------|------|
| 用户类型 | 系统中的用户角色分类，包括管理员、新人、主持人 |
| 账号映射 | 统一管理不同类型用户登录认证的表 |
| 逻辑删除 | 数据不真正删除，只标记is_deleted状态 |
| 角色 | 权限的集合，一个用户可以拥有多个角色 |
| 权限 | 系统功能的访问控制单位 |
